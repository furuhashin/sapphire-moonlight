# Change Log

All notable changes to the "Sapphire Moonlight" extension are documented here.
This project adheres to [Keep a Changelog](https://keepachangelog.com/) and
[Semantic Versioning](https://semver.org/).

## [0.0.1] - 2026-06-23

### Added
- Initial release of **Sapphire Moonlight**, an accessible high-contrast dark theme.
- Signature palette: sapphire cyan functions, golden-yellow types, lavender keywords,
  soft-purple variables, sky-blue parameters, rose numbers/literals, lime strings.
- WCAG AA+ contrast on every syntax color (most AAA), with moderate chroma to reduce
  halation / eye strain, and color-blind-aware hue & lightness separation
  (checked under protanopia / deuteranopia / tritanopia).
- Template / interpolation placeholder highlighting (`${...}`, f-string `{...}`).
- Variable sigil coloring for PHP (`$`) and Ruby (`@` / `@@`).
- Markdown styling: headings, bold/italic, inline code, strikethrough, links, blockquotes.
